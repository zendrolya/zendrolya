<h3>🚀Динамические</h3>

📋<a href="https://zendrolya.github.io/TaskList/" target="_blank">Список задач</a> - традиционный проект, направленный на отработку базовых концепций React и JavaScript: JSX, props, useState, компоненты, условный рендеринг, методы обработки массивов (map, filter), обработчик событий, LocalStorage. Позволяет добавить задачу: ввести ее название, приоритет и дату; просмотреть список задач с возможностью сортировки по дате/приоритетам, а также удалить или отметить как выполненную (отобразится в списке выполненных задач). Имеется также возможность раскрывать и скрывать каждый раздел.

🧩<a href="https://github.com/zendrolya/graph-explorer" target="_blank">Graph Explorer</a> - крупный учебный проект, находящийся в активной разработке. Является редактором, позволяющим работать с графами. Более подробное описание, в том числе возможности, находятся в Readme.md репозитория.
